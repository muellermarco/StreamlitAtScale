# StreamlitAtScale



## Getting Started
A simple example of how atscale's python integration can be used to integrate your data into a web application.

# Prerequisites
secrets.toml 
```toml
atscale_user = ""
atscale_password = ""
atscale_host = ""
atscale_project_id_draft = ""
atscale_project_id_published = ""
```
